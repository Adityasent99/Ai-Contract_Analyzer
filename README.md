Contracto.AI — AI Contract Analyzer

AI-powered web app that analyzes contracts for security, privacy, and legal risks, and returns a structured JSON report with actionable insights.


Features

 - Risk Analysis — Detects security, privacy, and legal issues
 - Structured Output — Clean JSON (API-ready)
 - PDF Support — Upload or paste contract text (via PDF.js)
 - Government Scheme Suggestions — Relevant Indian schemes with links
 - Interactive UI — Tailwind-based risk dashboard



 Tech Stack
 
- Frontend: HTML, TailwindCSS, JavaScript, PDF.js
- Backend: Node.js, Express, CORS, node-fetch
- AI: Groq API (LLaMA 3.3 70B)



Setup: 

git clone https://github.com/Adityasent99/Ai-Contract_Analyzer.git
cd Ai-Contract_Analyzer/aiContrBackend
npm install
node server.js


Note: 

Not a substitute for legal advice
Requires API key + internet
Ensure strict JSON output from LLM
